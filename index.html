Github Migration

I played a key role in a significant GitHub migration project within our team, where our primary responsibility was to seamlessly transfer approximately 2700 organizational repositories from Bitbucket or GitLab to GitHub. This initiative aimed to streamline our development and deployment processes, leveraging GitHub's robust features for a more efficient and cohesive workflow.
